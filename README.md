# calculatormini
mini calculator / grapher inspired by desmos
Created for AP Computer Science A
6/11/2026

# Program Description
Graph any equation you want using the interactive app (with respect to x).
Input a function, submit it, and then click the graph button.

# Special Features
* Is able to plot almost any function (with respect to x, singular variable)
* Dyamnic input system (multiple functions can be plotted)
* Custom drawing system 
* Optimized for android phones
